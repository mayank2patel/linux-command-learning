# pwd (Present working directory)

In Linux, files and directories are organized in a *hierarchy* called the **filesystem**.

## The Directory Tree in Linux

The entire filesystem starts from a single top-level directory called the root directory, represented by a forward slash **/**.\
From the root, the directory tree branches into subdirectories, which can contain files and more subdirectories.

```plaintext
/
|-- bin
|   |-- file1
|   |-- file2
|-- etc
|   |-- file3
|   `-- directory1
|       |-- file4
|       `-- file5
|-- home
|-- var
```

## Understanding File Paths

A path that starts with / is an absolute path because it starts at the root directory.
